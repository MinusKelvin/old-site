Contemplating the 3rd Dimension
2016-01-25

I'm thinking of making my 2D tile game a 3D tile game since I've written a fast voxel renderer, but at that point it might just seem like a Minecraft clone. I've also settled on the working title "Tile Tech" for my game.

So now let's go over the pros and cons of using 3D for Tile Tech:

Pros

  • Only lighting algorithm fast enough is floodfill; lighting is really simple
  • Extra dimension makes building stuff far more flexible
  • Stuff like attacking is much simpler

Cons

  • Lighting is less awesome
  • Another dimension means much higher memory usage
  • Another dimension also means there are more things to check and so CPU usage is much higher
  • I have to make models for things
All in all I think the pros outweigh the cons, and since I've started plenty of 2D tile games it's a nice change of pace.
